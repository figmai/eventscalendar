# Weekly Calendar - React Application

A modern React application that implements a weekly date selection interface with intuitive controls and responsive design.

## Features

- **Today Button**: Quickly select the current week (Monday to Sunday)
- **Date Range Picker**: Interactive calendar with two-month view for easy date selection
- **Automatic Week Selection**: Any selected date automatically adjusts to show the full week
- **Weekly Cards Display**: Beautiful card-based layout showing all 7 days of the selected week
- **Current Day Highlighting**: Today's date is visually highlighted in the week view
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices
- **Accessible**: Keyboard navigation and ARIA labels for screen readers

## Tech Stack

- **React 18**: Modern functional components with hooks
- **Vite**: Fast build tool and development server
- **date-fns**: Lightweight date manipulation library
- **react-datepicker**: Customizable date picker component

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd weekly-calendar
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and visit the URL shown in the terminal

## Usage

1. **Select Current Week**: Click the "Today" button
2. **Choose Any Week**: Click the date picker to open the calendar
3. **View Week Details**: The 7 cards display each day of the selected week
